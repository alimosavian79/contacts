# Contacts

* How to run it?  
First run this command and wait for the packages to finish installing  
```  
npm install  
```  

Then run This command  
```
npm run serve
```  
now open your browser and go the port assigned for the project.  

for checking out the tests run
```
npm run test:unit
```
